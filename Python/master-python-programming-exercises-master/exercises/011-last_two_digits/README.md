# `011` last two digits

## 📝 Instructions:

1. Complete the `last_two_digits()` function. Given an integer greater than `9`, `last_two_digits()` prints its last two digits.

## Example input:

```py
last_two_digits(1234)
```

## Example output:

```py
34
```

# `016` century

## 📝 Instructions:

1. Write a function  `century()`. Given a year (as a positive integer), `century()` finds the respective number of the century. 

## Example input:

```py
century(2001)
```

## Example output:

+ 21

## 💡 Hints:

+ Note that, for example, 20th century began with the year 1901.
